# Icon Files Needed

This directory requires two icon files for the PWA to function properly:

## Required Icons

1. **icon-192x192.png**
   - Size: 192x192 pixels
   - Format: PNG
   - Purpose: Standard PWA icon for Android and other platforms

2. **icon-512x512.png**
   - Size: 512x512 pixels
   - Format: PNG
   - Purpose: High-resolution icon for splash screens and app listings

## Design Guidelines

- Use a simple, recognizable design
- Ensure the icon works well at small sizes
- Use contrasting colors for visibility
- Consider the theme color (#2196f3) when designing
- Make sure the icon is centered with appropriate padding

## Online Icon Generators

Create your icons using these free tools:

1. **Real Favicon Generator**: https://realfavicongenerator.net/
   - Upload a source image (at least 512x512)
   - Generate all required sizes automatically

2. **PWA Builder**: https://www.pwabuilder.com/
   - Provides PWA icon generator tool
   - Creates icons in multiple sizes

3. **Favicon.io**: https://favicon.io/
   - Generate from text, image, or emoji
   - Quick and simple interface

4. **Canva**: https://www.canva.com/
   - Design custom icons from scratch
   - Export in required sizes

## Quick Start Icon

For testing purposes, you can use a simple placeholder:

1. Create a 512x512 square in any image editor
2. Add a background color (e.g., #2196f3)
3. Add text or logo in the center
4. Export as PNG
5. Create a copy and resize to 192x192

## Suggested Icon Design

For the Local GPT PWA, consider:
- A brain or neural network symbol
- The letters "LG" or "GPT"
- A chat bubble with AI theme
- A notebook or document icon with tech elements
- Blue gradient background matching the theme color

## Validation

After adding icons, validate using:
- Chrome DevTools > Application > Manifest
- Check that icons appear correctly
- Test installation on mobile device
- Verify splash screen displays properly
